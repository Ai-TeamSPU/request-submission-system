# SQL: นำเข้าข้อมูล Excel ลงตาราง requests

## คำอธิบาย

- **ไฟล์ต้นทาง:** `teaching_no_checkin.xls` (471 แถว)
- **ตารางปลายทาง:** `requests`
- **การค้นหา email:** match ชื่ออาจารย์กับ `faculty.name_th` / `faculty.full_name_th` → ดึง `faculty.email`
- **ค่าเริ่มต้น:** problem_type = "ไม่ได้เช็คอิน", reason = "นำเข้าจากรายงานอาจารย์ไม่เช็คอิน"
- **Trigger:** `trg_create_approval` จะสร้าง `approvals` (status: Pending) ให้ทุกแถวอัตโนมัติ
- **ป้องกันซ้ำ:** ตรวจ duplicate ด้วย ชื่ออาจารย์ + รหัสวิชา + วันที่ + กลุ่ม

## วิธีใช้

1. เปิด Supabase Dashboard → SQL Editor
2. คัดลอก SQL ด้านล่างทั้งหมด
3. กด Run
